Create a React Greets User App

Overview

Objective is to create a simple react app that greets the user depending on the time of the day.

What's needed:

  • React app created using create-react-app npm package.
  • 2 JSX variables: time and greeting. time for storing the time of the day. And greeting for storing different greetings (good morning, good afternoon etc.) based on the time of the day.
  • JS Date object for getting the current time of the day.

How to Create the React Greets User App?

Follow the instructions given below:

  1. Clear out the div content of App component.
  2. Define 2 variables 'greeting' and 'time' above the return statement of the App component.
    • Leave variable greeting unassigned. Its value is determined depending on the time of the day.
    • Extract the current time (in hours) using the JS Date object and assign it to 'time' variable.
    • You can check whether time is being extracted correctly by console logging the time variable.

    Modifying value of the 'greeting' variable depending on the time of the day.

    Using if-else-if-else statement, we can now change the value of the 'greeting variable'.

    • If the time indicates morning, the greeting variable should display 'Good Morning'.
    • If the time indicates afternoon, the greeting variable should display 'Good Afternoon'.
    • If the time indicates night, the greeting variable should display 'Good Night'.
