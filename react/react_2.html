Simple 2 Component Layout Using React

Overview

Objective is to create a basic 2 component layout using react. One representing header. And another the body.

What's needed:

  • React app created using create-react-app npm package.
  • Modification of index.html, index.css and app.js files within the project folder.
  • Google font for styling the font in our react app.

Components are 'parts' of your react app. It's a bunch of html tags thrown together (with or without variables) that make up different sections of your app. In our simple app, we'll be using 2 of them. One for displaying the header (hello world) and another for displaying the body (para text).

To keep things simple, we will be displaying only one html element per component. In reality, a component can contain many html elements.

First we will:

  1. Create the HTML tags <h1> and <p>
  2. Style them and then
  3. Convert them into react header and body components.

React Component

React component is basically a custom html element (that encloses within itself several conventional html elements) which react allows its devs to create. Syntax-wise, they are similar to html self-closing tags like <img/> and <input/> box. Naming is left to our choice as long as react rules are followed. But they should always begin with a capital letter.

Components are of 2 types: functional and class. We will be dealing only with functional components here. Our simple app will have 2 components: <Header/> and <Body/>.

Our App.js itself is a component responsible for rendering the entire react app. Meaning all other sub components within the app - like the ones we are about to create - have to be placed inside the App component for them to be rendered on the browser. If you check out index.js file, you'll see that <App/> has been placed inside the ReactDOM.render() method.
