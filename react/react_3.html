Displaying Your Name Using JSX Variable In React

Overview

Objective is to store your name in a jsx variable and display the same on the web browser when the app runs, instead of hard coding it directly into the HTML.

What's needed:

  • React app created using create-react-app npm package.
  • JSX variable for storing your name.

JSX Variables

React combines HTML, CSS and Javascript into a single unit and allows us to roll out our apps. And since there are no separate spaces for our variables (the JS part) and markups (the HTML part) to reside in, they have to be embedded together. This sort of embedding is brought about by JSX.

JSX allows us to write javascript variables within HTML. And this simple app serves as a good example for it.

  • JSX variables have to be defined above the return statement of the component.
  • They are always wrapped within curly {} braces when inside the HTML.

Creating the JSX Variable

In the app.js file, do the following:

  1. Above the return statement of the App function, declare a const variable and assign your name to it.
  2. Within the return statement and inside the <div> insert the following html line: <h1> Hi, My Name is {name} </h1>. Make sure you have wrapped the jsx variable name (which contains your name) within curly {} braces.
  3. Save the file. Now you will see your name printed on the browser. Meaning the React DOM evaluated the value stored in the jsx variable and printed it on the screen.
