How to Create Multiple Divs?

Overview

The objective is to create 3 divs. Give each one of them a height and width and color them.

How to Create the Divs?

To create the divs, follow the instructions given below:

  1. Create a html page (basically a file with .html extension)
  2. Add 3 <div> </div> tags within the body section.
  3. Give all of these divs the id named 'box' using which they can be styled later.
  4. Write one, two and three as content for the 3 divs so that you can identify them easily.

The divs will appear on the browser as shown in the image below.

Adding Height, Width and Color to the Divs

  1. Within the <head> </head>section, add <style> </style> tag.
  2. Within the <style> </style> tag, add the following code.
      3. #box { border:1px solid black; height:200px; width:200px; background-color: red; }
  3. Save the file and open it using a web browser.

The divs would be much bigger in size now - but uniformly so - and would be colored red as shown here.

Why Didn't The Divs Align Side By Side?
This is because the div elements take up the entire space horizontally by default. That is, they don't allow other elements - even other divs - to sit beside them. Therefore whatever that follows has to be placed under the div and not beside it.

In CSS terminology, divs are known as 'block' elements. Block elements do not allow other elements to sit beside them. But this default behavior of divs can be modified. We will see how this can be done when we create an entire web page's layout later.
