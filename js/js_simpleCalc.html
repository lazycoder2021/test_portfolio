Simple Calculator Using JavaScript

Overview

Objective is to create a simple calculator that allows the user to enter 2 numbers and perform addition, subtraction, multiplicaton or division.

What's needed:

  • 2 Input boxes to allow the user to enter the numbers.
  • Select box that allows the user to select the operation to be performed.
  • Button to trigger the user selected operation (addition, subtraction, multiplication or division).
  • Alert box to show the result to the user.

How to Grab the User Inputs using Javascript?

Include your javascript code within <script> </script> tags just before </body> tag .

  1. Create variables num1, num2 for storing 1st and 2nd user inputs. Leave them unassigned for now.
  2. Create another variable op and leave it unassigned. This will store the operation which the user selects.
  3. Create another variable btn for storing the button element.
  4. Add an event listener to the button and set it up such that it grabs the numbers entered by the user along with the selected operation and assigns them to variables num1, num2 and op respectively (which were previously left unassigned) the moment the button gets clicked.

How to Execute the Appropriate Operation Based on User Selection?

Our calculator by default should support 4 operations: that is, addition, subtraction, multiplication and division. And based on the user selection (which as we have seen, gets stored in the variable op), the appropriate operation should get executed. How do we achieve this in JavaScript?

By making use of Switch condition statement.The Switch condition statement helps us to select one option from a list of options and is therefore perfect for our simple calculator where the user is expected to select one arithmetic operation from a list of operations.

As you can see, we pass the op variable (which contains the user selected arithmetic operation) to the switch statement and have listed all our available calcy operations within it. Now the switch statement compares the value of the op variable it receives with the operations listed within {} and executes the operation (or case) that matches the value stored within variable op.

Note that we are just console.logging the result for now for testing our program. Later, this can be replaced with an alert box to show the result to the user.

We can also make use of if-else condition statement to create the arithmetic operations for our calculator, though the Switch statement fits our program's requirements perfectly.
