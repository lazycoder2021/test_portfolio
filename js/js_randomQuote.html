Display Random Quote Using JavaScript

Overview

Objective is to select a quote from a list of available quotes and display it on the browser when the user clicks the 'Get Quote' button.

What's needed:

  • Div to display the selected quote on screen
  • Button to trigger the display quote operation.
  • Array to store the list of quotes
  • Math.random() to randomly select a quote from the available list of quotes.

How to Generate the Random Quote using Javascript?

Include your javascript code within <script> </script> tags just before </body> tag .

  1. First create an array variable called quotes and store within it all your quotes as strings.
  2. Use Math.random() method to generate an index number between 0 and 3 (the length of the quotes array) and then pass it to quotes (as quotes[index number]) variable to randomly generate a quote.

    4. As you can see, the function generateRandomQuote() picks/returns a random quote from quotes array every time the function runs. We now have to call this function inside the button's event listener. This will ensure that every time the button gets clicked a random quote gets generated.

  4. Since the generateRandomQuote() function returns a quote, we declare a variable called quote (local to the function) to capture the randomly generated quote.
  5. Now all we have to do is render the quote stored within the variable quote on screen. For this, we modify the innerHTML property of the div 'display' (which was previously empty) and assign it the value of the variable quote (the randomly picked quote).
