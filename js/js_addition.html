Adding 2 Numbers Using JavaScript

Overview

Objective is to allow the user to submit 2 numbers that will be added and the result displayed on screen.

What's needed:

  • 2 Input boxes to allow the user to enter the numbers to be added.
  • Button to trigger the addition operation once the user enters the numbers.
  • Alert box to show the result to the user.

ID attributes help us grab the html elements and manipulate them in javascript based on the user input. ID num1 grabs the first user input and num2 grabs the second. ID btn helps us grab the button element. Within it, we will include the formula for addition.This formula gets invoked when the user presses the button.

How to Grab the User Inputs using Javascript?

Include your javascript code within <script> </script> tags just before </body> tag .

  1. Create variables num1, num2 for storing 1st and 2nd user inputs. Leave them unassigned for now.
  2. Create another variable num3 and leave it unassigned. This will be responsible for storing the result later.
  3. Create another variable btn for storing the button element.
  4. Add an event listener to the button and set it up such that it grabs the numbers entered by the user and assigns them to variables num1 and num2 (which were previously left unassigned) the moment the button gets clicked.
  5. To check whether you are grabbing the numbers being input correctly, just console.log it on your browser as shown in the code below.

Now that you've grabbed the user entered numbers, next step is to add them and display the result.

How to Add the Numbers and Display the Result?

You just have to add the 2 numbers obtained from the user and store it in the variable num3.

  • num3 = num1 + num2;

Then display the result stored in the variable num3 using the alert box to the user.
