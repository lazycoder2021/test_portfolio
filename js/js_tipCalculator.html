Tip Calculator Using JavaScript

Overview

Objective is to create a simple tip calculator which takes the bill amount as the input and calculates the tip.

What's needed:

  • Input box to get the user input.
  • Select box which allows the user to select the tip percentage based on the type of service provided.
  • Button for performing the tip calculation.
  • Alert box for displaying the output.

How to Calculate Tip using Javascript?

Include your javascript code within <script> </script> tags just before </body> tag .

  1. First create variables for storing the bill amount and the tip percentage, leave them unassigned.
  2. Next create a variable for storing the button element.
  3. Add an event listener to the button and set it up such that it the values of billAmount and tipPercentage are captured the moment the user enters them.

    4. Try console logging the values to ensure you are grabbing them correctly.

  4. Now to calculate the tip, multiply the bill amount with the tip percentage and divide them by 100. Store the resulting value in a variable called tip (local variable).
  5. Use an alert box to output the calculated tip.
