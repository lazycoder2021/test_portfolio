Finding Out Whether a Given Number is Even or Odd

Overview

Objective is to allow the user to submit a number and check whether it is even or odd by dividing it by 2.

What's needed:

  • Input box to allow the user to enter the number to be checked.
  • Modulos operator (%) to check if the division produces a remainder or not.
  • Alert box to show the result to the user.

How to Grab the User Input using Javascript?

Include your javascript code within <script> </script> tags just before </body> tag .

  1. Create a variable num for storing the input value and leave it unassigned for now.
  2. Create another variable for storing the html button element.
  3. Add an event listener to the button and set it up such that it grabs the number the moment the user enters it in the input box and stores it in the variable num which was previously left unassigned.

How to Check if the Number is Odd or Even Using Modulus Operator?

% produces a remainder if the number is not divisible; produces 0 if the number is perfectly divisble.

% combined with if-else condition can help us find out if the user input is odd or even.
