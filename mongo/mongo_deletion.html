Deleting Databases, Collections & Documents From MongoDB

Overview

Objective is to delete a database, collection and a document from MongoDB.

Deleting a Database

Follow the instructions given below:

  1. In the mongo shell, access the database you wish to delete.
  2. Use dropDatabase() command to delete the database as shown in the image.
  3. The ok message confirms that the database has been successfully deleted.

Deleting a Collection from the Database

Follow the instructions given below:

  1. In the mongo shell, access the database from which you wish to delete a collection.
  2. Using drop() command you can delete a collection as shown here:
  3. 'true' message indicates the collection has been deleted successfully.

Deleting a Document from a Collection

Documents can be deleted in 2 ways: single document deletion and bulk deletion.

Single Document Deletion

  1. You can use remove() command and pass in the deletion creiteria between the brackets as shown here:
  2. Deletion criteria is nothing but the key:value pair using which you reference the document to be deleted.
  3. The removed message confirms the deletion of the document.

Bulk Document Deletion

  1. You can also choose to remove all the documents from a collection by using remove() command and passing an empty object as shown here:
  2. {} refers to all the documents within the collection.
  3. The removed message confirms the deletion of the documents.
