Creating Database Using MongoDB

Overview

Objective is to create a simple database using MongoDB via command line interface.

What's needed:

  • MongoDB installation

How to Create a Database using MongoDB?

After you have successfully installed MongoDB, follow the instructions given below:

  1. Press 
    Windows + R
    keys and type in 'mongo' into the box.
  2. MongoDB console (Mongo Shell to be more precise) will open up as shown below.
  3. Clear the screen (using 'cls' command) so that you can begin without any distraction.
  4. In the blank screen, type the following mongodb command: show dbs
    • This will list the databases which exist in Mongo by default.
  5. To create a new database, use the followig command: use [database name]
    • So let's create a 'users' database. The command would be use users.
    • 'switched to db users' message should get displayed.
  6. This means you have successfully created the 'users' database using MongoDB shell.
    • To confirm, type db in the shell.
    • db is the mongoDB command for displaying the database currently in use.
    • users will get displayed.

    Now try executing show dbs command again. You'll notice that the newly created database does not appear yet in the list of available databases. This is because the database does not contain any records at the moment.
