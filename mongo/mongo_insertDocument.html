Inserting Documents into a Collection Using MongoDB

Overview

Objective is to create and insert 2 documents into a collection using MongoDB shell.

Documents are data records written in key: pair values (just like JSON).

Documents are written in BSON language (JSON variant that is encodable).

How to Insert Documents into a Collection?

Follow the instructions given below:

  1. In the mongo shell, access the database into which you wish to add the 2 documents
  2. First you have to access the collection into which the documents are to be added as shown below:
  3. Next you have tack onto the collection, the insert command as shown below (using .)
  4. The document data to be inserted goes inside the brackets.
    • The format in which the data should be written is shown in the image below.
    • The format is same as formatting json data.
    • Every record goes within curly {} braces.
    • And every key:value pair is separated by a comma (',').
    • Both key and value should be wrapped within single or double quotes.
    • If the record was submitted successfully, you will get a success message.
  5. In case of inserting multiple documents: Multiple documents should be treated as array of objects.
    • That is, every document would be wrapped in curly {} braces
    • Seperated from each other by a comma (,)
    • And all of them in turn will be placed inside an array using [] brackets.
