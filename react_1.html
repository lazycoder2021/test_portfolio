Hello World Using React

Overview

Objective is to create a simple react app that prints the words 'hello world' on the browser.

What's needed:

  • Node Installation
  • NPM Package Installation

How to Create the Hello World React App?

We will use create-react-app npm package for creating our react app.

  1. Open up command line interface and cd into the folder where you want create your react app.
  2. Type the following command:npx create-react-app myfirstreactproject

    3. myfirstreactproject is the name of the react app we are going to create.

  3. Once the commend successfully executes itself, you will be instructed to do the following.
  4. As suggested, cd into the myfirstreactproject folder and then execute npm start command.
  5. React app you just created will automatically open up on your browser on the port 3000.
  6. Your react app will look like this.

    7. This is the default screen for all react apps. All we have to do is modify it so that it displays 'hello world'.

  7. A js file by name 'app.js' is responsible for displaying the default screen.
  8. app.js file can be found in myfirstreactproject folder within the src sub folder.
  9. Open the app.js file. It will look like as shown below.
  10. Remove everything written in-between <div className="App"> and </div>
  11. Replace them with <h1> Hello World </h1>. Save the file.
  12. Your react app will now be displaying the words 'Hello World' on the browser.
