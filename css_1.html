What are DIVs in CSS?

Overview

Divs are basically square shaped blocks that help you decide the webpage's layout or design. Every element you see on the browser is a div with a certain height and a width. Divs can also be shaped differently. They can even be turned into perfect circles. By default, though, they appear as squares or boxes on the screen.

How to Create a Div?

To create a div, follow the instructions given below:

  1. Create a html page (basically a file with .html extension)
  2. Add <div> </div> tags within the body section.
  3. Add some text between the tags like this: <div> Good Morning</div>
  4. Save the file and open it using a web browser.

That's it! You just successfully created a div - the building block of web design.

Your div will not appear as a box, but simply as the text you wrote within <div> </div>. This is because divs aren't visible by default. You have to add style to them to make them visible.

How to Add Style to a Div?

Styling refers to the cosmetic modifications done to change appearance of otherwise plain HTML elements.

To add style to a div, follow the instructions given below:

  1. Within the <head> </head>section, add <style> </style> tag.
  2. Within the <style> </style> tag, add the following code.
      3. div { border:1px solid black; }
  3. Save the file and open it using a web browser.

You will notice that your text is now wrapped within a rectangular box as wide as the width of your browser.

What exactly happened?
You just gave your div a border of 1px thickness and colored it black. In css the styling has to go within {} braces. And the property and its value should be separated by a ':' Every property:value pair should be terminated by a ';''

How to Postion a Div?

Your div will appear a bit misplaced at the moment. To fix this, you have to position your div correctly.

To position your div, follow the instructions given below:

  1. Add 'text-align:center' property to the div's styling code within <style> </style> tags.
  2. Save the file and open it using a web browser.

Technically, we just positioned or aligned the text within the div and not the entire div.
