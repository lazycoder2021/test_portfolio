Exploring a Free Weather API

The best way to start learning about apis is to explore one on your own.

A good place to start would be https://openweathermap.org/ . Open weather API, as it is popularly known as, provides a bunch of apis that provide information on the weather conditions of places across the globe. Now if you've ever woundered how some browsers display your local weather conditions using little widgets, the answer is using apis like these.

Open weather api has a free edition you can use for testing the api. But first, you have to subscribe to the service. On you register yourself, you will be provided with an API KEY using which you can try out their apis.

But before you do I would suggest installing the JSON formatter chrome extension on your browser, because this will simplify the appearance of the response.

Trying Out Open Weather API

Login to the website using your credentials and,

  1. Click 'API' on the navigation menu.
  2. Under the Current Weather Data section, click API doc.
  3. Observe some 'Example of API Calls' and their responses, get a feel of the api.
  4. Finally, try the first one (getting weather condition By city name) out on your own.
  5. It's quite simple. Copy the link and replace the API KEY placeholder with your API KEY value, and city name placeholder with the name of a valid city as shown in the image.
  6. Now paste the url in your browser and observe the response you get.

What Did You Just Do?

You performed a simple REST API GET operation and got the resource you were looking for.

Observe the Response Carefully and Understand It

Most important of all would be the main section - which displays max temp, min temp, feels like, humidity - that is filled with data characteristic of weather widgets. Therefoee has been given the name 'main'.
