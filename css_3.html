Aligning Divs Side By Side

Overview

The objective is to create 3 divs with equal height and width and place them next to each other. This sort of arrangment plays a crucial role in creating webpage layouts.

How to Place the Divs Side by Side?

There are 2 ways to achieve this. The best method is to make use of the CSS property called 'flex'.

  1. Create a html page (basically a file with .html extension)
  2. Add 3 <div> </div> tags within the body section.
  3. Style these divs usign their tag name (that is, div).

    4. Use the following CSS.

    div { border:1px solid black; height:200px; width:200px; }

The divs will appear on the browser as shown in the image below.



About CSS Display Property
CSS display property is just that. It helps you display the elements in certain manner. It can also be used to hide the element, if required. The display property can take 4 different values: block, inline-block, flex and none (none hides the element altogether). For now, we'll just look at flex.

Make Use of the Flex CSS Property

For these divs to appear side by side, they should be treated as one single element (or one giant div).That is, all 3 of them should be wrapped in a parent div (also known as the wrapper or container div).

  1. Create a div with id="container" and move the previously created 3 divs into this div.
  2. Now set the CSS display property of the div with id="container" as flex as shown in the image.
      3. #container { display:flex; }
  3. Save the file and open it using a web browser.

The divs will appear on the browser as shown in the image below.You'll notice that the divs now appear side by side, although the widths of the divs might get affected a bit.

You will probably need to increase the width of the divs to make it look good again. But our job is done: we have successfully placed 3 divs side by side.

What Did Flex Property Do?
This is because the div elements take up the entire space horizontally by default. That is, they don't allow other elements - even other divs - to sit beside them. Therefore whatever that follows has to be placed under the div and not beside it.

In CSS terminology, divs are known as 'block' elements. Block elements do not allow other elements to sit beside them. But this default behavior of divs can be modified. We will see how this can be done when we create an entire web page's layout later.
